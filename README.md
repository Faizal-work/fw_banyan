
# FW_Banyan

## Introduction

FW Banyan is named after the Banyan tree - like the Banyan tree vines itself which grows by extending itself outwards and becoming new roots, this project is the revamped revamped consolidation of a decade of work and testament to my own growth as a developer.

Like the Banyan tree, it starts from one point and spreads outward.
Sometimes a butterknife is all you need. With loads of WD-40.

## Overview

The modules associated with this project:

| Module | Description |
|---|---|
| `Banyan.Core` | Auth, middleware, shared infrastructure |
| `Banyan.Trees` | Urban tree management with geospatial data |
| `Banyan.Analytics` | Derived data, scoring, and risk analysis |
| `Banyan.Notifications` | Event-driven alerts via RabbitMQ |
| `Banyan.Documents` | Report generation and PDF output |
| `Banyan.Entertainment` | Inflight entertainment system — media, games, e-shop |


## Tech Stack

- **Backend:** .NET 8, C#, Minimal API
- **Databases:** PostgreSQL + PostGIS, MongoDB, MSSQL, Redis
- **Messaging:** RabbitMQ
- **Frontend:** React, TypeScript
- **DevOps:** Docker, GitHub Actions, Azure
