
# FW_Banyan

## Introduction

FW Banyan is named after the Banyan tree. In short, like the Banyan tree vines itself, this project is the revamped version of all of my life's work. I want to improve my current code, remember those that I made and create an upgraded version of it as a POC and testament to my own growth as a developer.

Not the sharpest tool in the shed, but definitely the tool that you need in that moment. Sometimes, a butterknife is all you need. With loads of WD-40.

## Overview

The modules associated with this project:
1. Banyan.Trees -> Urban tree management
2. Banyan.Analytics -> Data management and scoring
3. Banyan.Notifications -> Alerts
4. Banyan.Documents -> Report generation
5. Banyan.Entertainment -> Airline management system
